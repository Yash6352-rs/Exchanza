For installing eas build

For your phone only, do this once:
    npm install -g eas-cli
    eas login
    eas build:configure

Then build APK:
    eas build --platform android --profile preview

Wait ~10-20 minutes.
    Expo will give you a URL.
    Open URL on phone → Download APK → Install.

After that:
    If you change code
    Build again:
    eas build --platform android --profile preview
    and install the new APK.

Faster updates later (optional)
    If you set up EAS Update:
    eas update
    Then users (you) get updates without reinstalling APK.
